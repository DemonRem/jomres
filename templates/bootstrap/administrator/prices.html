Please do not purchase any "Jomres Zero" license if you're not prepared to read the manual, or simply because it's the cheapest possible option. Jomres Zero licenses are intended for experienced web site developers who are on a tight budget. Jomres is a powerful system and if you purchase a Zero license you will need to be prepared to use the forum, the manual or your own ingenuity to answer your questions. We will not provide you with support when you realise you need hand-holding after installing it.

We offer more "Pre-purchase/pre-commitment" options than any of our competitors, because we want you to be sure that Jomres is right for you. Before committing to a purchase you have the option to install Jomres Core for free into any existing installation of Joomla or Wordpress. Additionally, if you request a (free) trial license key you are able to install one of the 4 Quickstarts to experiment with. By the time you're ready to purchase you should know if you are going to need support or not.
If you think you will need support, please purchase a license that includes support.
